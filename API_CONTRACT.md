# API Contract — KPubData Studio

### Datasets
- list datasets
- inspect dataset metadata
- preview sample records

### Build Specs
- load spec
- save/export spec
- validate spec

### Builds
- execute build
- get build status
- list build runs
- get manifest

### Artifacts
- list artifact outputs
- preview artifact contents

### Publish
- list publish targets
- trigger publish
- get publish status
