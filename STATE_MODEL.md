# State Model — KPubData Studio

## Draft State
Editable but not yet executed.

Statuses:
- `new`
- `dirty`
- `validated`
- `invalid`

## Build Run State
Execution lifecycle.

Statuses:
- `queued`
- `running`
- `succeeded`
- `failed`
- `cancelled`

## Publish State
Publication lifecycle.

Statuses:
- `not_started`
- `ready`
- `publishing`
- `published`
- `publish_failed`
