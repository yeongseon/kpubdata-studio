# 화면 설계서 — 애플리케이션 셸 (Layout Shell)

## 화면 개요

모든 페이지를 감싸는 공통 레이아웃 셸입니다. 좌측 사이드바 내비게이션, 상단 헤더, 테마 선택기, 모바일 오버레이 메뉴를 제공하여 화면 간 이동과 전역 컨트롤을 담당합니다.

- **컴포넌트**: `src/app/Layout.tsx`
- **적용 범위**: 모든 라우트(`src/app/router.tsx`의 최상위 레이아웃)

## 주요 UI 구성요소

| 영역 | 구성요소 | 설명 |
| :--- | :--- | :--- |
| 사이드바 | 로고 | `K` 마크 + `KPubData Studio` 텍스트 |
| 사이드바 | 주 내비게이션 | 대시보드 · 빌드 · 새 빌드 · 결과물 · 설정 (`primaryNavItems`) |
| 사이드바 | 테마 선택기 | `system` / `light` / `dark` 선택 `select` |
| 헤더 | 타이틀 | "Builder 제어 화면" + 부제 "학생과 메인테이너를 위한 공공데이터 빌드 워크플로" |
| 헤더 | 컨텍스트 CTA | 현재 경로에 따라 달라지는 액션 버튼(`headerCtaFor`) |
| 모바일 | 오버레이 메뉴 | 햄버거 토글, 배경 오버레이, `ESC` 닫기 |

## 상태 및 상호작용

- 활성 내비게이션 항목은 `bg-accent-subtle` 배경으로 강조됩니다(`NavLink` active).
- 상단 헤더는 `sticky`로 스크롤 시 고정됩니다.
- 모바일 뷰포트에서는 사이드바가 오버레이로 열리며, 오버레이 클릭 또는 `ESC` 키로 닫힙니다.
- 테마 선택 값은 `data-theme` 속성으로 반영되어 Tailwind `dark:` 변형을 활성화합니다.

## 접근성

- `ESC` 키로 모바일 메뉴를 닫을 수 있습니다.
- 내비게이션은 시맨틱 링크(`NavLink`)로 구성되어 키보드 이동이 가능합니다.
- 활성 항목은 색상뿐 아니라 배경 대비로도 구분됩니다.
