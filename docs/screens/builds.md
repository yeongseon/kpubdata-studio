# 화면 설계서 — 빌드 목록 (Builds)

## 화면 개요

지금까지 만든 모든 빌드를 한 표에서 검색·정렬·조회하는 화면입니다. 각 빌드의 상태와 마지막 실행 시각을 확인하고 상세 화면으로 이동합니다.

## 라우트 및 진입·이탈

| 항목 | 내용 |
| :--- | :--- |
| 라우트 | `/builds` |
| 컴포넌트 | `src/pages/BuildsPage.tsx` |
| 진입점 | 사이드바 "빌드", 대시보드 빠른 시작 |
| 이탈점 | 행 액션 → [빌드 상세](build-detail.md), "새 빌드" → [새 빌드](new-build.md) |

## 주요 UI 구성요소

| 구성요소 | 설명 |
| :--- | :--- |
| 제목 검색 | `TextInput`으로 빌드 제목 필터링 |
| 정렬 토글 | 시작 시각 기준 오름/내림차순 전환 |
| 빌드 테이블 | 열: 빌드 · 상태(`StatusBadge`) · 마지막 실행 · 액션 |
| 상태 뷰 | 로딩 / 오류(`ErrorState`+재시도) / 빈 목록 / 목록 |

## 상태 및 상호작용

- **로딩**: 데이터를 불러오는 동안 로딩 표시.
- **오류**: `ErrorState`와 재시도 버튼 노출.
- **빈 목록**: 검색 결과 또는 데이터가 없을 때 안내.
- **목록**: 검색어·정렬 상태에 따라 실시간 필터링.

## 데이터 소스

- `listBuilds()` (MOCK 모드에서는 데모 시드 데이터). 검색·정렬은 클라이언트에서 수행됩니다.

## 접근성

- 정렬 토글은 버튼으로 조작 가능하며 현재 정렬 방향을 표시합니다.
- 상태는 배지 색상과 텍스트를 함께 제공합니다.
