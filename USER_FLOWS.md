# User Flows — KPubData Studio

## 1. New Build Wizard
- choose starter template or blank
- add source
- configure params
- preview source
- choose exports
- fill metadata
- validate
- run build

## 2. Existing Build Review
- open previous build
- inspect manifest
- inspect files
- rerun with changes

## 3. Publish Review
- choose completed build
- verify dataset card
- confirm destination
- publish
